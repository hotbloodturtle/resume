## Introduce
이력서에 대한 몇 가지 불편한 마음을 개선하고자 마크다운 이력서 표준화 프로젝트를 시작합니다.

1. 개발자를 잘 표현할 이력서 양식이 없다.  
2. 2,3 년에 한번씩 업데이트해야 하는 이력서 작성할 때 너무 힘들다.
3. 이력서 양식이 도구에 너무 의존적이다.
